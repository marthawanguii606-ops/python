## EXPLORATION 1: Titanic Survival Analysis

This project analyzes the Titanic dataset to uncover survival patterns.
### Key Findings:
-**Children ( under 18)** had a **53%** survival rate
-**Adults (over 18)** had a **36%** survival rate
-Women survived at significantly higher rates than men
-Missing age data was handled using mean imputation (filled with ~30 years)
